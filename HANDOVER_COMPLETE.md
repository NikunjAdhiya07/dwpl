## 🔄 Complete Workflow Example

```
Step 1: Setup Masters
  → Add Party (with charges)
  → Add RM Items
  → Add FG Items
  → Create BOM (FG → RM mapping)
  → Add GST rates
  → Add Transport details

Step 2: Receive Material
  → Create GRN
  → Select party
  → Add RM items with quantities
  → Submit
  ✅ RM Stock increases automatically

Step 3: Process & Dispatch
  → Create Outward Challan
  → Select party
  → Select FG size (RM auto-fills from BOM)
  → Set annealing/draw counts
  → Enter quantity
  → Rate auto-fills from party
  → Charges auto-calculate
  → Submit
  ✅ RM Stock decreases, FG Stock increases
  ✅ Export 3-copy PDF

Step 4: Invoice
  → Create Tax Invoice
  → Select challan (all data auto-fills)
  → GST auto-calculates
  → Submit
  ✅ Export 3-copy PDF
```
